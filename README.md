# A Snakemake-based metagenomic data analysis

This is a Snakemake-based metagenomic data analysis pipeline. The pipeline is designed to be used with the [MetaPhlAn4.1](https://github.com/biobakery/biobakery/wiki/MetaPhlAn-4.1) and [HUMANnN3.9](https://github.com/biobakery/biobakery/wiki/humann3/) software packages.
The pipeline accomplishes the following tasks: read quality control, host filtering, taxonomic profiling, functional profiling, and functional annotation.
