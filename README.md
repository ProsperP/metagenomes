# A Snakemake-based metagenomic data analysis

This is a Snakemake-based metagenomic data analysis pipeline. The pipeline is designed to be used with the [MetaPhlAn4](https://github.com/biobakery/MetaPhlAn4) and [HUMANnN3.9](https://github.com/biobakery/humann3) software packages.
The pipeline accomplishes the following tasks: read quality control, host filtering, taxonomic profiling, functional profiling, and functional annotation.
